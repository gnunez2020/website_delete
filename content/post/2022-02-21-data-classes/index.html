Classes

##The class attribute relates to the S3 class of an object which determines its behaviour ###Examples: factors, dates, and data frames

Factors

##R uses factors to handle categorical variables, variables that have a fixed and known set of possible values

x <- factor(c("BS", "MS", "PhD", "MS"))
x
## [1] BS  MS  PhD MS 
## Levels: BS MS PhD
typeof(x)
## [1] "integer"
class(x)
## [1] "factor"
glimpse(x)
##  Factor w/ 3 levels "BS","MS","PhD": 1 2 3 2
as.integer(x)
## [1] 1 2 3 2

Dates

##We can think of dates like an integer (the number of days since the origin, 1 Jan 1970) and an integer (the origin) glued together

y <- as.Date("2020-01-01")
y
## [1] "2020-01-01"
typeof(y)
## [1] "double"
class(y)
## [1] "Date"
as.integer(y)
## [1] 18262
as.integer(y) / 365 # roughly 50 yrs
## [1] 50.03288
